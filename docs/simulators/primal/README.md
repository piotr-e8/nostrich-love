# Primal Simulator

## Overview

Primal simulator is planned for future development. Primal is a fast, modern web-based Nostr client with excellent UX and Bitcoin-themed design.

## Status

**Planned** - Not yet implemented

## About Primal

**Real Client**: [primal.net](https://primal.net)

### Key Features (Planned for Simulation)
- Web-based interface
- Orange Bitcoin-themed design
- Fast performance
- Marketplace support
- Rich media handling

## Platform

- **Type**: Web client
- **Primary Color**: Orange (#F97316)
- **Design**: Modern web, responsive

## Planned Features

Based on Primal's feature set:

1. Direct Messages
2. Zaps

4. Search
5. Relay Management
6. Badges
7. NIP-05 Identity
