# Snort Simulator

## Overview

Snort simulator is planned for future development. Snort is a minimalist web-based Nostr client known for simplicity and speed.

## Status

**Planned** - Not yet implemented

## About Snort

**Real Client**: [snort.social](https://snort.social)

### Key Features (Planned for Simulation)
- Minimalist web interface
- Teal accent color
- Fast and lightweight
- Core Nostr features

## Platform

- **Type**: Web client
- **Primary Color**: Teal (#14B8A6)
- **Design**: Minimalist, clean

## Planned Features

Based on Snort's feature set:

1. Direct Messages
2. Zaps
3. Threads/Replies
4. Search
5. Relay Management
6. NIP-05 Identity
7. Mute Lists

## Implementation Notes

When implementing:
- Focus on simplicity
- Minimal UI elements
- Fast load times
- Core features only
