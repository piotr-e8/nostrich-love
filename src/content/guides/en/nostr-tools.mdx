## Key Management Tools

### Key Converters & Generators

**Nostr Tools (key converter)**
- **URL:** [nostr-tools.com](https://nostr-tools.com)
- **What it does:** Convert between npub/hex/nsec formats
- **Why use it:** Useful for developers or when you need different key formats

**Nostr Key Generator**
- **What it does:** Generate new key pairs offline
- **Why use it:** Create keys in a secure, air-gapped environment

### Backup & Recovery

**Key Storage Best Practices**
- Password managers (Bitwarden, 1Password)
- Hardware security keys (YubiKey)
- Paper backups (written and stored safely)
- Encrypted USB drives

<Callout type="warning">
  **Security Note:** Never store your nsec (private key) in plain text files, email, or cloud storage without encryption.
</Callout>

**Video Nostr**

## Identity & Verification

### NIP-05 Providers

**NostrPlebs**
- **URL:** [nostrplebs.com](https://nostrplebs.com)
- **What it does:** NIP-05 identifiers (you@nostrplebs.com)
- **Cost:** Free tier available, paid plans ~$12-15/year
- **Best for:** Getting a human-readable identifier quickly

**NostrCheck**
- **URL:** [nostrcheck.me](https://nostrcheck.me)
- **What it does:** NIP-05 + additional verification tools
- **Features:** Free tier, extra utilities

**GetAlby**
- **URL:** [getalby.com](https://getalby.com)
- **What it does:** Lightning wallet + NIP-05
- **Best for:** Combining payments and identity

### Verification Tools

**Nostr.guru**
- **URL:** [nostr.guru](https://nostr.guru)
- **What it does:** Explore Nostr events and verify NIP-05
- **Features:** Event viewer, profile lookup, relay inspection

## Lightning & Zaps

### Wallets

**Alby** (Browser Extension)
- **URL:** [getalby.com](https://getalby.com)
- **Best for:** Desktop/web users
- **Features:** Browser integration, Nostr Wallet Connect

**Phoenix** (Mobile)
- **URL:** [phoenix.acinq.co](https://phoenix.acinq.co)
- **Best for:** Mobile users wanting self-custody
- **Features:** Non-custodial, automatic channel management

**Wallet of Satoshi** (Mobile)
- **Best for:** Complete beginners
- **Features:** Easiest setup, Lightning address included

**Zeus** (Mobile)
- **URL:** [zeusln.app](https://zeusln.app)
- **Best for:** Power users, node operators
- **Features:** Connect to your own node, advanced features

**Mutiny** (Web/Mobile)
- **URL:** [app.mutinywallet.com](https://app.mutinywallet.com)
- **Best for:** Browser-based self-custody
- **Features:** No app store needed, federated custody

### Zap Tools

**ZapSplits**
- **What it does:** Automatically split zap revenue
- **Best for:** Collaborative content (podcasts, team projects)

**Zap Stream**
- **What it does:** Real-time zap notifications
- **Best for:** Live events, streams
