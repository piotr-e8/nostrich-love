## Key Management Tools

### Key Converters & Generators

**Nostr Tools (key converter)**
- **URL:** [nostr-tools.com](https://nostr-tools.com)
- **What it does:** Convert between npub/hex/nsec formats
- **Why use it:** Useful for developers or when you need different key formats

**NAK (Nostr Army Knife)**
- **URL:** [nak.nostr.com](https://nak.nostr.com)
- **What it does:** Command-line tool and web interface for Nostr operations
- **Features:** Key generation, event signing, relay interaction, debugging
- **Best for:** Developers and power users who need CLI tools

**Nostr Key Generator**
- **What it does:** Generate new key pairs offline
- **Why use it:** Create keys in a secure, air-gapped environment

### Backup & Recovery

**Key Storage Best Practices**
- Password managers (Bitwarden, 1Password)
- Hardware security keys (YubiKey)
- Paper backups (written and stored safely)
- Encrypted USB drives

<Callout type="warning">
  **Security Note:** Never store your nsec (private key) in plain text files, email, or cloud storage without encryption.
</Callout>

## Identity & Verification

### NIP-05 Providers

**NostrPlebs**
- **URL:** [nostrplebs.com](https://nostrplebs.com)
- **What it does:** NIP-05 identifiers (you@nostrplebs.com)
- **Cost:** Paid only - 12,500-65,000 sats/year (pricing based on username length)
- **Note:** No free tier available
- **Best for:** Getting a human-readable identifier quickly

**NostrCheck**
- **URL:** [nostrcheck.me](https://nostrcheck.me)
- **What it does:** NIP-05 + additional verification tools
- **Features:** Free tier, extra utilities

**GetAlby**
- **URL:** [getalby.com](https://getalby.com)
- **What it does:** Lightning wallet + NIP-05
- **Best for:** Combining payments and identity

## Privacy & Security

### Signer Apps

**Primal Signer (iOS)**
- **Built into Primal app**
- **What it does:** Secure key storage with biometric authentication
- **Features:** FaceID/TouchID protection, no separate app needed
- **Best for:** iOS users who want built-in security

**Amber (Android)**
- **Dedicated signer app for Android**
- **What it does:** Secure key signing with approval prompts
- **Features:** Multiple key support, detailed signing requests
- **Best for:** Android users wanting maximum security

**Other Signer Apps**
- Various dedicated apps for secure key signing
- Keep your nsec off your main device
- Search your app store for "Nostr signer"

**Tor Support**
