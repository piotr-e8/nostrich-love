# Placeholder Avatars

## Overview
This directory contains 50 gradient-based placeholder avatars for use in the Nostr Client Simulator.

## Files
- `avatar-1.svg` through `avatar-50.svg`

## Design
Each avatar features:
- Unique gradient background (randomly generated warm colors)
- Generic person silhouette (head and shoulders)
- Semi-transparent white overlay for depth
- Circular crop

## Usage
Use these avatars for:
- Mock user profiles in simulations
- Testing avatar display components
- Placeholder content during development
- Consistent visual identity across demos

## Alternative Sources
For production avatars, consider:
1. **DiceBear** - https://dicebear.com (free avatar API)
2. **Pravatar** - https://pravatar.cc (placeholder images)
3. **UI Faces** - https://uifaces.co (real user photos)
4. **Random User Generator** - https://randomuser.me

## Generation
These avatars were generated with randomized color gradients to ensure visual variety while maintaining a cohesive aesthetic.
