## Key Features

### Security
- Encrypted key storage
- Passphrase protection
- Memory zeroing
- No browser tech

### Performance
- Rust-based
- LMDB database
- OpenGL rendering
- Fast startup

### Privacy
- Tor support
- Privacy options
- No avatar loading option
- No image loading option

### Content Moderation
- Lists
- Muting
- Thread dismissal
- Content warnings
- Spam filtering scripts
- SpamSafe relay designation

## Unique Differentiators

### Gossip Model
- Advanced relay selection
- NIP-65 based
- Dynamic relay management
- Efficient connection handling

### No Web Stack
- Pure native implementation
- No JavaScript
- No HTML/CSS
- Reduced attack surface
